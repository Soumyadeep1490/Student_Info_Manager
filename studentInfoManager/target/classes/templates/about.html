About Student Information Manager

Here users can register with their First Name, Last Name, Username and Password. They can login using their Username and password to see the details they have entered. To change any details or delete the account, students have to contact with Administrators.

Administrators have special access to see own profile as well as check students' profiles. They can Edit and Delete their own accounts as well as students' accounts.

In this project, I have used Java Programming Language, Spring boot framework, HTML5, css3. The Maven Dependencies I have chosen are - spring-boot-starter-data-jpa, spring-boot-starter-thymeleaf, spring-web, spring-boot-starter-web, spring-boot-devtools, mysql-connector-java, validation-api, hibernate-validator.
